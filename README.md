# Sales Data Analysis & Dashboard Project

## Objective
Analyze sales data to identify trends and provide business insights.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel

## Key KPIs
- Total Sales
- Total Orders
- Average Order Value
- Top Category

## Analysis
Performed segmentation analysis based on:
- Category
- Region
- Customer Segment

## Dashboard
Created an interactive dashboard in Power BI.

## Hypothesis Testing
Tested whether Technology category generates higher sales than Furniture using T-test.

## Conclusion
Technology category contributes the highest revenue and business should focus on high-performing segments.
